# Sublime_Settings
How to set up sublime in linux/mac/windows for the first time for python and java developers.

### The Following commands should be followed:
```
ctrl+shift+p          #command pallette
install package control
Package Control: Install Package
    a. Predawn  : Color Scheme
    b. MaterialTheme:  Theme
    -> In Preferences Tab open settings
    -> In user paste and save the contents of "subl_pref_settings.txt" file(Make necessary changes of where the python is)
    Re-Open Sublime(After Each Installation)
    c. BracketHighlighter
    d. SideBarEnhancemets
    e. Anaconda
      -> Settings_user->Paste the contents of "anaconda_subl_settings.txt" file(after making the necessary changes about the location of python)
   * You can make a new build system by making change in the proper position of the file and saving it with the desired name but no changes in the extention is to be done whatsoever. File:"buildsys.txt"
    f. Emmet
    g. GitGutter
    h. Javatar
    i. JDebug
    j. Formatter
    k. SublimeLinter
    l. View in Browser
    m. Live Reload
    n. Minify
    o. SublimeCodeIntel
    p. JavaIME
    q. JavaImports
    r. Kite : Download and Install from the site -> "https://kite.com/download/"
 For more packages refer to this site: "http://aslanbakan.com/en/blog/33-essential-sublime-text-plugins-for-all-developers/"

```
